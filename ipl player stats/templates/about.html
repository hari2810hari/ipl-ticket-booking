About IPL Player Stats

Your ultimate destination for comprehensive Indian Premier League statistics, match fixtures, and ticket booking services.

Our Mission

To provide cricket enthusiasts with a seamless platform to access player statistics, match details, and ticket booking services for the Indian Premier League.

Key Features

Player Statistics

Detailed player profiles with comprehensive statistics including batting averages, strike rates, bowling figures, and more for all IPL teams.

Match Fixtures

Complete schedule of all IPL matches with venue information, timings, and team matchups. Never miss an important game again.

Ticket Booking

Easy-to-use ticket booking system with seat selection for your favorite matches. Secure your spot in the stadium hassle-free.

Technology Stack

Flask

Python web framework used for building the backend of this application.

MySQL (XAMPP)

Relational database management system for storing all IPL data.

HTML5 & CSS3

Modern web technologies for creating responsive and interactive user interfaces.

JavaScript

Used for dynamic content and interactive features throughout the application.
